# Overview

## What the module does

**wp-module-staging** provides staging functionality for WordPress sites in Newfold brand plugins: create staging site, clone production to staging, and restore from staging. It registers with the Newfold Module Loader; the host plugin provides the UI and any REST endpoints that call the module.

## Who maintains it

- **Newfold Labs** (Newfold Digital). Distributed via Newfold Satis.
